STEP 1 — Video Upload & Processing

User uploads video from phone.

Backend flow:

Save file temporarily

Use FFmpeg to:

Extract audio (.wav)

Extract key frames (every 0.5–1 second)

Example command:

ffmpeg -i input.mp4 -vn -acodec pcm_s16le -ar 44100 -ac 2 output.wav


And:

ffmpeg -i input.mp4 -vf fps=1 frames/output_%04d.png


Now you have:

Audio file

Image frames
