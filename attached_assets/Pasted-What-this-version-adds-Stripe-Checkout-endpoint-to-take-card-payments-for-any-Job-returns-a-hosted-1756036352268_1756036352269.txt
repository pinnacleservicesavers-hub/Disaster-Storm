What this version adds

Stripe Checkout endpoint to take card payments for any Job (returns a hosted payment link).

Dropbox Sign endpoint to send Contract/AOB for e-signature (supports test mode).

Lien rules by state (editable) and an API that computes the deadline from completion date.

Keeps everything from v1 (NWS alert texts, SPC damage pins, GPS “Report Damage,” AI letter).

How to use the new features
A) Take a card payment (Stripe)

Click Create Payment Link (temporary UI) → enter Job ID, description, and amount.

It opens a Stripe Checkout page. When paid, you’ll get a checkout.session.completed webhook (endpoint added at /webhooks/stripe) that writes a paid invoice row and a payment artifact.

In production, go to Stripe → Developers → Webhooks → point to https://YOUR_URL/webhooks/stripe and paste the signing secret into STRIPE_WEBHOOK_SECRET. (The included route shows where to enable raw-body verification.)

B) Send Contract/AOB for e-signature (Dropbox Sign)

POST to /api/sign/send with:

{
  "job_id": "UUID",
  "customer_name": "Jane Homeowner",
  "customer_email": "jane@example.com",
  "subject": "Emergency Tree Service Contract",
  "message": "Please review and sign.",
  "contract_pdf_url": "https://your-public-contract.pdf",
  "aob_pdf_url": "https://your-aob.pdf"
}


The API sends the signature request (test mode if DROPBOX_SIGN_TEST=1) and logs an artifact.

Add a webhook in Dropbox Sign → point to /webhooks/dropbox-sign. When all parties sign, we record a SIGNED artifact; you can map back to the job with the stored signature_request_id.

C) Lien deadlines (state-specific + editable)

The table lien_rules(state, days, notes, require_prelim) is included.

Edit or add a state via:

POST /api/lien/rules with { "state":"AL", "days":90, "notes":"Residential typical", "require_prelim": false }

Compute a deadline:
GET /api/lien/deadline?state=GA&completion_date=2025-08-01
→ returns the calculated date (defaults to 90 days if rule missing).

You’ll later set each Job’s lien_state and completion_date; a daily cron can text nudges as the deadline approaches.
