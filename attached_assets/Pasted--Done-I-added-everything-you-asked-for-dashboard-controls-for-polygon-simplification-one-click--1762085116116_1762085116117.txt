New features (ready now)
1) Dashboard controls

Simplification slider to set polygon simplification tolerance.

Buttons to ingest MRMS Hail / Precip / Flood on-demand with your chosen tolerance.

Bulk SMS button to notify contractors in hazard areas.

Open: /web/dashboard/index.html

2) MRMS on-demand API

POST /api/mrms/run?peril=hail|heavy_precip|flood_risk&tolerance=0.02
Calls your MRMS ingestor for the selected peril, applying the tolerance live.

3) Bulk SMS (Moratorium)

POST /api/notify/moratorium-sms
Body:

{ "message": "Moratorium ON: Stage outside hazard polygon.", "only_hazard_contractors": true }


Sends to contractors inside active hazard areas (detected via geometry intersection). Uses your Twilio creds if present; otherwise logs.

4) FNOL PDF reports

GET /api/reports/fnol/{claim_id}
Streams a clean PDF including claim + carrier + asset and intersecting hazards at the asset’s location. Perfect to share with insurers/customers.
