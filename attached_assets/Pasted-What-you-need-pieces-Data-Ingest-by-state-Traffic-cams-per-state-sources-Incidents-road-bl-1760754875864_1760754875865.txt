Unified data model
// Common shapes your UI/alerts use

type Camera = {
  id: string;
  state: string;          // "FL","GA",...
  name: string;
  city?: string | null;
  lat: number;
  lon: number;
  imageUrl?: string | null;
  videoUrl?: string | null;
  source: "IBI"|"ArcGIS"|"Scrape";
  updatedAt: string;
};

type Incident = {
  id: string;
  state: string;
  type: "ACCIDENT"|"ROAD_CLOSED"|"HAZARD"|"CONSTRUCTION"|"WEATHER"|"OTHER";
  description: string;
  lat: number;
  lon: number;
  startTime?: string;
  endTime?: string;
  lanes?: string | null;
  severity?: number | null;   // 1–5 if provided
  source: "511"|"ArcGIS"|"Scrape";
  updatedAt: string;
};

type WeatherAlert = {
  id: string;  // NWS CAP id
  event: string;   // "Tornado Warning", "High Wind Warning", ...
  state: string;
  headline: string;
  description: string;
  effective: string;
  expires?: string;
  polygon?: Array<[number,number]>; // lat,lon pairs
  areas: string[];   // counties/UGC
  source: "NWS";
};

Where to pull the data

GA: 511 GA (you already have GA511_KEY) → cameras + incidents.

SC: 511 SC (same vendor style as GA; use your SC511_KEY when you get it).

FL: FL511 cameras via ArcGIS FeatureServer (public). Incidents also available via ArcGIS or FL511.

NC: DriveNC/NCDOT – cameras via ArcGIS feature layer; incidents via TIMS services/ArcGIS.

TN: SmartWay – events via ArcGIS; cameras page is HTML (scrape to JSON via your PROXY_URL).

AL: ALGO – site provides cameras; use scraping/proxy to JSON unless you find ArcGIS.

Severe Weather (US-wide): NWS api.weather.gov/alerts (CAP/GeoJSON) filtered by event (e.g., Tornado Warning, High Wind Warning) and by area (state) or bounding box.

Wind conditions: NWS gridpoints (/gridpoints/{office}/{x},{y}) or observations (/stations/{id}/observations/latest).

You don’t have to finish all states to launch. Start with FL + GA + SC, then add NC, then TN/AL via proxy.

Pollers (server)

Create a tiny Node service (Express + cron) that:

Polls each data source every 60–120s.

Normalizes to the common shapes above.

Stores latest into your DB and pushes deltas to the app via WebSocket/SSE.

Example: NWS Tornado/High-Wind alerts (US-wide or by state)

// server/pollers/nws.js
import fetch from "node-fetch";

export async function fetchNwsAlertsForState(state2) {
  // Examples of events you care about:
  const events = [
    "Tornado Warning",
    "Tornado Emergency",
    "Severe Thunderstorm Warning",
    "High Wind Warning"
  ];
  const url = `https://api.weather.gov/alerts?status=actual&message_type=alert&limit=200&area=${state2}&event=${encodeURIComponent(events.join(","))}`;
  const res = await fetch(url, { headers: { "User-Agent": "ContractorApp/1.0 (contact@example.com)" }});
  if (!res.ok) throw new Error(`NWS ${res.status}`);
  const json = await res.json();
  // Map to WeatherAlert
  return (json.features || []).map(f => {
    const p = f.properties || {};
    return {
      id: f.id,
      event: p.event,
      state: state2,
      headline: p.headline,
      description: p.description,
      effective: p.effective,
      expires: p.expires,
      areas: p.geocode?.UGC || [],
      polygon: (p.geocode?.SAME ? null : (p.areaDesc?.includes("...") ? null : null)), // you can also parse p.geometry
      source: "NWS"
    };
  });
}


Example: FL cameras via ArcGIS (public)

// server/pollers/florida.js
const FL_CAMERAS_URL = "https://services.arcgis.com/3wFbqsFPLeKqOlIK/ArcGIS/rest/services/FL511_Traffic_Cameras/FeatureServer/0/query?where=1%3D1&outFields=*&f=json";

export async function fetchFloridaCameras() {
  const res = await fetch(FL_CAMERAS_URL);
  if (!res.ok) throw new Error(`FL cams ${res.status}`);
  const json = await res.json();
  return (json.features || []).map(ft => {
    const a = ft.attributes || {};
    const g = ft.geometry || {};
    return {
      id: String(a.OBJECTID ?? a.Id ?? a.ID),
      state: "FL",
      name: a.Name || a.Title || a.Description,
      city: a.City || null,
      lat: g.y ?? a.Latitude,
      lon: g.x ?? a.Longitude,
      imageUrl: a.StillImageUrl || a.Url || null,
      videoUrl: a.VideoUrl || null,
      source: "ArcGIS",
      updatedAt: new Date().toISOString(),
    };
  });
}

export async function fetchIbiCameras(base, key, state2) {

    source: "IBI",
    updatedAt: new Date().toISOString(),
  }));
}

Alert rules (start simple, grow later)
// Pseudocode
for each WeatherAlert:
  if event in ["Tornado Warning","Tornado Emergency"]:
     notifyContractorsNearby(polygon or county, radius=50mi, priority="CRITICAL")

for each Incident:
  if text contains ["power line", "lines down", "tree on home", "tree on house", "vehicle collision", "disabled vehicle", "car in roadway"]:
     notifyContractorsNearby(point, radius=20mi, priority="HIGH")

for each Camera frame (optional future):
  if human/mod flags “tornado/debris/power lines down” on a cam:
     attach cam snapshot URL + location and notify


Dedupe & cooldown

Don’t spam: same incident/alert → suppress re-notif for 30–60 min unless severity/area changes.

Per-contractor throttle: max X criticals per hour.

Distance & geo

Use a small helper (Haversine) + fast point-in-polygon (if you use NWS polygons). Geo-index your contractors.

function haversineMi(a, b) {
  const R = 3958.8;
  const toRad = (d)=>d*Math.PI/180;
  const dLat = toRad(b.lat-a.lat);
  const dLon = toRad(b.lon-a.lon);
  const lat1 = toRad(a.lat), lat2 = toRad(b.lat);
  const h = Math.sin(dLat/2)**2 + Math.cos(lat1)*Math.cos(lat2)*Math.sin(dLon/2)**2;
  return 2*R*Math.asin(Math.sqrt(h));
}
