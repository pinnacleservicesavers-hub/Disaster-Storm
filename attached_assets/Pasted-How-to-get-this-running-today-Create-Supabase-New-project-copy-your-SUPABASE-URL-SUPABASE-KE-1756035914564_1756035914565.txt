How to get this running (today)

Create Supabase

New project → copy your SUPABASE_URL & SUPABASE_KEY.

SQL editor → paste supabase_schema.sql from the zip and run it.

Create your keys (free/cheap tiers)

Twilio: phone number for SMS/voice → grab TWILIO_SID, TWILIO_TOKEN, and your TWILIO_NUMBER.

LocationIQ (free) for reverse-geocode: LOCATIONIQ_KEY.

OpenAI: OPENAI_API_KEY.

(Later) Stripe (payments) + Dropbox Sign/DocuSign (e-sign) + Estated/ATTOM (property ownership).

Import to Replit

New Replit → Node.js → upload the zip and extract.

Add Secrets:
SUPABASE_URL, SUPABASE_KEY, TWILIO_SID, TWILIO_TOKEN, TWILIO_NUMBER,
OPENAI_API_KEY, LOCATIONIQ_KEY, and optional COVERED_STATES (e.g., GA,AL,FL,...).

Shell: npm install → npm run dev.

Open the webview: you’ll see the map + a Report Damage button.

Seed one subcontractor
In Supabase → subcontractors → Insert a row with your name/phone, sms_opt_in = true, states = {"GA","AL","FL",...}.

Test alerts
The worker in server/alertWorker.js runs automatically. When NWS posts a Tornado/Severe T-Storm/Hurricane alert for one of your states, you’ll get a text.

Recommended stack & why (picked for you)

Supabase (auth, Postgres, Storage, real-time): fast dev + exportable data.

Leaflet + OSM tiles (map): zero license headache; swap in Mapbox later if you want styling.

NWS/NOAA + SPC + NHC: official, free, reliable.

Twilio: SMS/voice/call recording/IVR on one platform.

Dropbox Sign (or DocuSign): standard e-signature.

Stripe: simplest payments + receipts.

OpenAI: clean letters, rebuttals, weekly claim summaries.

Guardrails (important)

Public Adjusting: in many states only licensed PAs can negotiate claims—keep your AI as a drafting assistant; you (or a licensed PA) presses “Send.”

Call Recording: play a consent message and offer non-recorded option in two-party states.

Messaging: give adjusters/homeowners an opt-out (“Reply STOP”).

Property Data: use licensed APIs (Estated/ATTOM), not scraping or people-finder sites.

Reviews: you can moderate what shows on your website/app; you can’t hide third-party platform reviews.
