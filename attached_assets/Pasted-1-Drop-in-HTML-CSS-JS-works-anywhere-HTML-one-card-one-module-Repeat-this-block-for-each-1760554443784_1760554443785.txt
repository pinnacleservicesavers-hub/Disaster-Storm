JS (lazy-inject the video and only play when visible)
<script>
(function(){
  const cards = document.querySelectorAll('.mod-card');

  const loadVideo = (card) => {
    if (card.dataset.loaded) return;
    card.dataset.loaded = "1";
    const wrap = card.querySelector('.mod-media');
    const v = document.createElement('video');
    v.muted = true; v.playsInline = true; v.autoplay = true; v.loop = true;
    v.setAttribute('preload','metadata');
    // Sources
    const s1 = document.createElement('source'); s1.src = card.dataset.videoWebm || card.dataset.videoWebm; s1.type = 'video/webm';
    const s2 = document.createElement('source'); s2.src = card.dataset.videoMp4  || card.dataset.videoMp4;  s2.type = 'video/mp4';
    v.appendChild(s1); v.appendChild(s2);
    // Fallback image (optional)
    if(card.dataset.poster) v.poster = card.dataset.poster;
    wrap.appendChild(v);
  };

  const io = new IntersectionObserver(entries=>{
    entries.forEach(e=>{
      if(e.isIntersecting){ loadVideo(e.target); }
    });
  },{rootMargin:"200px 0px"});

  cards.forEach(c=>io.observe(c));
})();
</script>


How it works:

You keep your exact card markup.

Set data-video-mp4/data-video-webm per card.

JS lazily injects <video> only when the card is near viewport (great for performance).

Dark gradient overlay keeps text readable on any footage.

2) WordPress/Divi quick steps

Upload your short background clips to Media Library (create /media/ directory if you prefer). Provide both WebM and MP4. Aim for 6–12 seconds, 720p, ~1–4 MB each (HandBrake: H.264, Constant Quality ~22; WebM VP9 similar size).

In Divi: use a Code module for each card (paste the HTML block above).

Put the CSS in Divi → Theme Options → Custom CSS (or your divi-child2/style.css).

Paste the JS once site-wide in Theme Options → Integration → “Add code to the <body>” (or enqueue via child theme).

For mobile, the lazy-load + overlay already keeps it light; if you need to fully disable video on mobile, add:

@media (max-width: 640px){ .mod-media video{ display:none } }
