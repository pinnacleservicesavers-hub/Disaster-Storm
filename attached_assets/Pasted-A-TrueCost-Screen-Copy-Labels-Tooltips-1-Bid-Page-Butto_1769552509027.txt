E) Default Calculations (What to compute on the backend)
Labor cost per labor item

wage = qty * (reg_hours * base_rate + ot_hours * base_rate * ot_multiplier)
burden = wage * (burden_percent/100)
per_diem = qty * per_diem_per_day * planned_days (or exact days)
lodging = qty * lodging_per_night * (planned_days-1 or planned_days)
labor_item_total = wage + burden + per_diem + lodging + travel_allowance

Equipment item total

If hourly: rate * hours_per_day * days_used
If daily: rate * days_used
If weekly: rate * ceil(days_used/7)
Plus:
fuel_cost + (maintenance_reserve_per_hr * hours_per_day * days_used) + (insurance_alloc_per_day * days_used) + mobilization_one_time

Materials

base = unit_cost * quantity
waste = base * (waste_factor_percent/100)
materials_item_total = base + waste + tax_shipping

Overhead

If fixed: amount
If percent_of_revenue: bid_revenue * (amount/100)
If per_day: amount * planned_days

Totals

direct_costs = labor + equipment + materials
contingency = direct_costs * (contingency_percent/100)
total_job_cost = direct_costs + overhead + contingency
net_profit = bid_revenue - total_job_cost
net_margin_percent = net_profit / bid_revenue * 100
