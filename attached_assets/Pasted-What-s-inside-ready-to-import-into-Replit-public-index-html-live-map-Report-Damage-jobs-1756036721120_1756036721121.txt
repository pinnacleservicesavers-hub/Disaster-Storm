Quick use

Supabase → SQL: run supabase_schema.sql, then supabase_lien_seed.sql.

Create Storage bucket named artifacts (make public).

Replit: npm install → npm run dev.

Subcontractors: insert your phone in subcontractors with sms_opt_in=true.

Visit /jobs:

Create a job → Stripe Payment button opens a hosted checkout.

Send Contract/AOB button triggers Dropbox Sign (paste your public PDF URLs).

Upload photos/videos → saved in Storage and logged as artifacts.

Lien reminders: set lien_state and completion_date on the job; SMS goes out at the intervals (10/7/3/1 days remaining).

Notes & best practices

Property: /api/property/lookup?address=... uses Estated (licensed, ToS-safe).

Call recording: when you add Twilio voice, play a consent notice and respect two-party states.

Claim “nudge”: /api/claim/nudge drafts the email; you approve/send (keeps you clear of PA restrictions).
