How to use in Replit (Node.js example)

In Replit → Secrets (Environment Variables)
Add:

AZURE_TTS_KEY

AZURE_TTS_REGION

Install axios:

npm install axios


Use this code (generates an MP3 file):

import fs from "fs";
import axios from "axios";

const key = process.env.AZURE_TTS_KEY;
const region = process.env.AZURE_TTS_REGION;

async function speak(text) {
  const url = `https://${region}.tts.speech.microsoft.com/cognitiveservices/v1`;

  const ssml = `
  <speak version='1.0' xml:lang='en-US'>
    <voice xml:lang='en-US' xml:gender='Female' name='en-US-JennyNeural'>
      ${text}
    </voice>
  </speak>`;

  const response = await axios({
    method: "post",
    url,
    headers: {
      "Ocp-Apim-Subscription-Key": key,
      "Content-Type": "application/ssml+xml",
      "X-Microsoft-OutputFormat": "audio-16khz-128kbitrate-mono-mp3"
    },
    data: ssml,
    responseType: "arraybuffer"
  });

  fs.writeFileSync("voice.mp3", response.data);
  console.log("Saved voice.mp3");
}

speak("Hello John. This is a free AI voice running from Replit.");


✅ Change voices by swapping the voice name:

Female: en-US-JennyNeural, en-US-AriaNeural

Male: en-US-GuyNeural, en-US-DavisNeural
