Title: Weather module shows stale/incomplete guidance—must include ZIP/County, latest SPC/NWS timing windows, and wind gust guidance

Summary:
The app displays a generic “marginal risk” note but does not (1) localize by ZIP and County, (2) surface latest SPC/NWS timing windows, or (3) include forecast wind gust guidance for the user’s location. This leads to operational decisions being made on stale or non-actionable data.

Environment:

App: Pinnacle Service Savers (production)

Host: Replit

Affected module/page: Severe Weather Advisory card

Example location: ZIP: <YOUR_ZIP> — County: <YOUR_COUNTY, STATE>

Steps to Reproduce:

Open the Severe Weather Advisory card for <YOUR_ZIP>/<YOUR_COUNTY>.

Compare the text against the latest NWS/SPC products for that county.

Note lack of localized timing windows (e.g., Friday 1–9 PM, Saturday afternoon → overnight) and missing max wind-gust guidance.

Actual Result:

Generic “Marginal (1/5)” text with no ZIP/County tags, no time windows, no gust values.

Expected Result:
Card should render location-aware, time-boxed guidance with:

ZIP & County clearly printed at top.

SPC outlook level + timing windows for that county (e.g., “Fri 1–9 PM; Sat PM → overnight”).

NWS forecast wind gust range for that ZIP (e.g., “Gusts 35–45+ mph possible”).

Timestamp (UTC/local) + product source tags (SPC, NWS Forecast Office, issuance time).

“Contractor Ops” checklist (staging triggers, safety).

Impact / Why this matters:
We stage crews, materials, and drones. Without county-level timing and gusts, we risk under- or over-deploying resources, safety exposure, and missed SLAs.

Requested Fix (Acceptance Criteria):

Inputs: ZIP → resolve County + Lat/Lon.

SPC: Fetch current Day 1–3 outlook for county polygon; extract risk level and valid time window.

NWS Local Forecast: Pull forecast discussion + hourly/period gusts for the ZIP’s point.

Render:

Header: “Severe Weather – <ZIP> | <County, ST> (as of <timestamp>).”

Body bullets: Risk level, When, Threats (damaging winds / hail if applicable), Wind gust guidance (range with units), Confidence note.

Footer: Source badges (SPC, NWS), issuance times, and link to warnings feed.

Failover: If any API fails, show a yellow fallback banner: “Live data unavailable—check NWS/SPC links.”

Data Sources (standard, public):

SPC Outlook GeoJSON/KML (Day 1–3).

NWS API (points → forecast & hourly) for gusts + forecast discussion.

County lookup via FCC or Census API; ZIP→county mapping as cache.
