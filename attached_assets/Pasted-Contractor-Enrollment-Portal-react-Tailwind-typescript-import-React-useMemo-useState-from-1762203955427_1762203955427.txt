I built you a contractor enrollment portal component that does exactly what you asked:

Contractors can enter company name, owner names, address, phone, email, website.

They can upload public-facing photos/videos for customers to view (with a visibility toggle).

They can upload business license, insurance, and multiple certifications.

There’s a live “Profile Score” that ranks them higher as they add: license, insurance, certifications, complete contact details, and review links.

Fields to paste Google review link, a social review page (e.g., Facebook/Yelp), and an in-app review link (so customers can be texted/emailed a review page).

Clean, modern UI styled with Tailwind; ready to drop into your Replit React app.

Open the canvas on the right to see the full React code.

How to use it in your app

Make sure Tailwind is set up in your Replit project (Vite or Next.js).

Create a route/page and import the component:

import ContractorEnrollment from "./ContractorEnrollment";
export default function Page() { return <ContractorEnrollment />; }


Hook up the POST /api/contractors endpoint on your backend to save:

companyName, owners[], address, phone, email, website

media[] (public photos/videos)

documents: businessLicense, insurance, certifications[]

reviewLinks: google, social, inApp

score (0–100) and rank (Bronze/Silver/Gold/Platinum)

Minimal backend shape (example)

If you’re using Node/Express, create /api/contractors to accept JSON and persist it (Prisma, Mongo, etc.). The component already fetches to that URL.

Adjusting the ranking

Inside the component there’s a SCORE_WEIGHTS object you can tweak:

Business License: +15

Insurance: +15

Certifications: +10 (+ small bonus per extra)

Contact details, website, media, and review links also add points.

Rank bands: Bronze <40, Silver 40–59, Gold 60–79, Platinum 80–100 (editable).
