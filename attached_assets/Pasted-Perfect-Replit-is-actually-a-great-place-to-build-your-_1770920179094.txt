STEP 1 — Install Dependencies

In Replit:

pip install fastapi uvicorn openai ffmpeg-python psycopg2 requests python-multipart


You’ll also need to install FFmpeg in your Replit environment.

STEP 2 — Video Upload Endpoint

In FastAPI:

Create /upload

Accept video file

Save temporarily

Call video_processor

STEP 3 — Extract Audio + Frames

In video_processor.py:

Use FFmpeg:

Extract audio:

ffmpeg -i video.mp4 -vn output.wav


Extract frames:

ffmpeg -i video.mp4 -vf fps=1 frame_%04d.png


Now you have:

Audio file

5–10 image frames

That’s enough for AI reasoning.

STEP 4 — AI ANALYSIS (Simplified for MVP)

You do NOT need custom sound ML yet.

Instead:

Send:

Audio file transcript or description prompt

3–5 image frames

To OpenAI Vision model with structured prompt:

Prompt example:

“Analyze engine noise and visible engine components in these frames. Identify abnormal sounds, vibration, smoke, fluid leaks, belt misalignment, or knocking. Provide top 3 likely mechanical issues with probability percentages.”

Return JSON format only.
