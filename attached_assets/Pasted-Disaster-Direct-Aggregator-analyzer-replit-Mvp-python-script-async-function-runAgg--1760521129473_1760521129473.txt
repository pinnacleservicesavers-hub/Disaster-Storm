Boom — all wired up. I upgraded your Replit-ready app in the canvas to include:

Premium feed hook (Tomorrow.io): class added; it auto-enables when TOMORROWIO_API_KEY is present.

Parcel upload + street-level impact: upload GeoJSON parcels via /parcels, and the app intersects event footprints with parcels using shapely. Impacted parcels come back in /aggregate and /risk-score.

Simple dashboard at / (HTML+JS): run queries, see events, upload parcels, and view impacted parcels without leaving the browser.

Quick start

In Replit, create requirements.txt with:

fastapi==0.115.0
uvicorn==0.30.6
httpx==0.27.2
pydantic==2.9.2
shapely==2.0.6


Paste the canvas code into app.py.

(Optional) Set TOMORROWIO_API_KEY in the Replit Secrets panel.

Run:

uvicorn app:app --host 0.0.0.0 --port 8000 --reload


Open the webview to use the Dashboard:

Try Dallas defaults, click Run to see the risk score + events.

Upload a GeoJSON parcel file and re-run to see Impacted Parcels.
